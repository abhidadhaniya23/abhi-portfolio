## Introduction

This is a [Movie App](https://react-movie-app-000.netlify.app/) built with React.js and TailwindCSS. It uses the [OMDB API](https://www.omdbapi.com/) to search for movies and display their details.

## Purpose of the project

- To learn how to use and implement concepts of React.js
- To learn how to use [TailwindCSS](https://tailwindcss.com/)
- To learn how to use the OMDB API
- To learn animations using [Framer Motion](https://www.framer.com/motion/)

## Features

- Search for movies
- View movie details
- View movie cast
- View movie reviews and rating
- View movie trailers
