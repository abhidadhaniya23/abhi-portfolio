## Introduction

[GigaResources](https://gigaresources.xyz) is a manually curated list of resources for developers, designers and creators. It's a collection of tools, libraries, articles, and more. It's a great place to find new tools and resources for your next project.

We are also running up a [newsletter](https://gigaresources.xyz) where we send out a weekly digest of the best resources we've found.

## Features

- **Search** - Search for resources by name, description, or tags
- **Filter** - Filter resources by tags
- **SSR** - Server-side rendering for fast page loads
- **News Letter** - Weekly newsletter with the best resources we've found
